# MIBs-ER4
```ER4_MIBs.txt``` contain full -probably- MIB, to be specific **HOST-RESOURCES-MIB**
(Because ER4 is cut-down Debian OS) list for SNMP monitoring on **EdgeRouter 4**, with MIB names and OIDs.

Works with any NMS tool like **Nagios** (which I'm using).

**Tools used:** ```snmptranslate```, ```snmpwalk```, AI (sorting, correction)

**Note** 
For entries like: 
```IP-FORWARD-MIB::ipCidrRouteDest,.1.3.6.1.2.1.4.24.4.1.1[.X.X.X.X.M.M.M.M.N]```
the part in brackets `[.X.X.X.X.M.M.M.M.N]` just shows the **instance values** for each table entry (in that case - routes).  
- These values are **not part of the MIB itself** — they’re just generated by the device for each row.  
- When documenting or copying OIDs, you can safely **ignore the bracketed part** and just use:
```.1.3.6.1.2.1.4.24.4.1.1```.
